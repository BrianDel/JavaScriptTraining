Exercise3

Using what you have learned from the Modern Javascript course, upgrade the todo list to use the new language features in ES6.
 - Template Strings
 - Arrow functions
 - Classes/inheritance
 - let and const declarations
 - Object destructuring

Add documentation to your classes.
 - Add jsdoc to your environment
  - npm install -g jsdoc3
 - Write proper documentation for your classes and methods.
  -http://usejsdoc.org/
 - Generate the jsdocs, e.g. jsdoc *.js
